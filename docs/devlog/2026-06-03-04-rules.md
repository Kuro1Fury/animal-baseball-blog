## Strike / Out 计数

在 GameDirector 里添加了两个计数变量：

```gdscript
var strike_count: int = 0
var out_count:    int = 0
```

三振触发出局：

```gdscript
func _register_strike() -> void:
    strike_count += 1
    if strike_count >= 3:
        _register_out("Strike Out!")
    else:
        _show_result("Strike %d!" % strike_count)
```

三个 Out → 半局结束：

```gdscript
func _register_out(result_text: String) -> void:
    out_count += 1
    if out_count >= 3:
        _show_result(result_text + "\nHalf Inning Over!")
        _reset_timer = 2.5
```

## 垒位占用（`b6fe0f7`）

```gdscript
var first_base_occupied: bool = false
var first_base_runner         = null
```

击球者跑到一垒且判定 Safe 后：

```gdscript
func _handle_safe_occupancy() -> void:
    if not first_base_occupied:
        _runner_1b.global_position = _first_base_point.global_position
        _runner_1b.visible         = true
        first_base_occupied        = true
        first_base_runner          = _runner_1b
```

Bug `1e471e2`：Runner1B 节点有碰撞体，导致跑者被卡在一垒旁。修复方法是在 `_ready()` 里禁用碰撞：

```gdscript
_runner_1b.get_node("CollisionShape2D").set_deferred("disabled", true)
```

`set_deferred` 不是立刻执行，而是在当前帧物理处理结束后执行，避免物理引擎冲突。
